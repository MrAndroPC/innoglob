1)Лицензия на оказание услуг связи по передаче данных для целей передачи голосовой

3)Лицензия на оказание услуг подвижной радиотелефонной связи;
4)Лицензия на оказание услуг связи по передаче данных, за исключением услуг связи по
передаче данных для целей передачи голосовой информации.
