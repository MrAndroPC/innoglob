Общий размер начисленных (подлежавших выплате) доходов по ценным бумагам эмитента

эмитента (общий размер дивидендов, начисленных на акции эмитента определенной

Общее количество ценных бумаг эмитента (количество акций эмитента соответствующей

Дата, на которую определялись лица, имевшие право на получение доходов, выплаченных
по ценным бумагам эмитента: 13 октября 2017 года.
Дата, в которую обязательство по выплате доходов по ценным бумагам эмитента
(дивиденды по акциям, доходы (проценты, номинальная стоимость, часть номинальной
стоимости) по облигациям) должно быть исполнено, а в случае, если обязательство по
выплате доходов по ценным бумагам должно быть исполнено эмитентом в течение
определенного срока (периода времени), дата окончания этого срока: не позднее 20 ноября
2017 года.
Общий размер доходов, выплаченных по ценным бумагам эмитента (общий размер
дивидендов, выплаченных по акциям эмитента определенной категории (типа): 14 182 541
162,40 рублей.
В случае если доходы по ценным бумагам эмитента не выплачены или выплачены
эмитентом не в полном объеме, причины невыплаты доходов по ценным бумагам эмитента:
- не выплачены дивиденды в адрес ПАО АФК "Система" на сумму 6 600 612 272,80 рублей
по причине ограничения права ПАО АФК "Система" на получение дохода по арестованным
ценным бумагам ПАО "МТС" (на основании Постановления о возбуждении исполнительного
производства в отношении ПАО АФК "Система" от 26.06.2017г.);
- не выплачены дивиденды на сумму 5 168,80 рублей в связи с тем, что на дату выплаты
акции находятся на счету эмитента в рамках реализации процедуры обязательного выкупа
акций (ст.75 ФЗ об АО);
- не выплачены дивиденды на сумму 9 776,00 рублей по причине непредставления
акционерами реквизитов для перечисления дохода.
